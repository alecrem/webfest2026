# レシピ集 (すきなものを ためしてみよう)

エディタの左側にコードがある。**1行だけ書きかえて → 💾 保存して表示 (Ctrl+S)**
を押すと、右のプレビューがすぐに変わる。むずかしく考えず、まず変えて、見て、
またもどす — の くりかえしでOK。

上から順にやらなくてよい。気になったものを、すきなだけ試そう。
こまったら 🗑️ **最初にもどす** を押せば、テンプレートに戻れる。

---

## 1. ポケモンの番号を変える

```html
<poke-card
  src="/api/pokemon/25"
  color="#ffde00"
  title="お気に入りポケモン"
></poke-card>
```

`25` の数字を **1〜1025** の好きな番号に変えてみよう。全然ちがうポケモンが出る。

---

## 2. 色を変える

`color="#ffde00"` の `#ffde00` の左に小さな四角 (色スウォッチ) がある。
クリックすると色えらびが開くので、好きな色を選ぶ。16進の文字は自動で
書きかわるので、覚えたり計算したりしなくてOK。

`<style>` の中の色 (背景の色・文字の色・タイトルの色など) にも同じ
色スウォッチが付いている。コメントを見れば、それが何の色か分かる。

---

## 3. 見出し (title) を変える

```html
<poke-card
  src="/api/pokemon/25"
  color="#ffde00"
  title="お気に入りポケモン"
></poke-card>
```

`title="お気に入りポケモン"` の文字を、自分の好きな見出しに変えてみよう。
カードの上の色帯に、その文字が出る。

---

## 4. 自分の部活に変える

テンプレートには最初 `kitaku` (帰宅部) が入っている:

```html
<club-list
  src="/api/bukatsu/kitaku"
  color="#16a34a"
  title="私の部活"
></club-list>
```

`kitaku` を、下の表から自分の部活の id に変えてみよう。

| id          | 部活               |
| ----------- | ------------------ |
| `yakyu`     | 野球部             |
| `soccer`    | サッカー部         |
| `volley`    | 女子バレーボール部 |
| `basket`    | バスケットボール部 |
| `kendo`     | 剣道部             |
| `rikujo`    | 陸上部             |
| `suisogaku` | 吹奏楽部           |
| `bijutsu`   | 美術部             |
| `kitaku`    | 帰宅部             |

---

## 5. 同じデータを、ちがう聞き方でもらう

バックエンドは、同じ種類のデータでも「アドレスの書き方」で答えが変わる。
部活 (bukatsu) で3つの聞き方を試してみよう:

- 1件だけ: `/api/bukatsu/kitaku`
- ぜんぶの一覧: `/api/bukatsu`
- 条件でしぼった一覧: `/api/bukatsu?category=文化部`

右上の「**バックエンドをのぞく →**」で、この3つのアドレスを順番に送信して、
返ってくるデータがどうちがうか見くらべてみよう。多くのバックエンドは、
こんなふうに「1件」「一覧」「しぼりこみ (`?○○=`)」を用意している。

気に入ったアドレスは、そのまま `<club-list>` の `src="..."` に貼ればOK。

---

## 6. 掛け持ちの部活をまとめて出す

2つ以上の部活に入っている人は、`ids` にカンマ区切りで並べると、
1枚のカードにまとめて出せる:

```html
<club-list
  src="/api/bukatsu?ids=yakyu,bijutsu"
  color="#2f4135"
  title="私の部活"
></club-list>
```

同じやり方は、ポケモンやチームでも使える:

```html
<poke-card src="/api/pokemon?ids=25,6,9" title="お気に入り"></poke-card>
<team-card
  src="/api/npb/teams?ids=hawks,giants"
  title="応援しているチーム"
></team-card>
```

---

## 7. 応援しているチームを変える

```html
<team-card src="/api/npb/teams/hawks" title="応援しているチーム"></team-card>
```

`hawks` を別の球団に変えてみよう (下の表の id、または一覧 `/api/npb/teams`
で確認できる)。サッカーが好きな人は `npb` を `jleague` に変える:

```html
<team-card
  src="/api/jleague/teams/kashima"
  title="応援しているチーム"
></team-card>
```

| npb id (プロ野球) | チーム                   | jleague id (Jリーグ) | チーム           |
| ----------------- | ------------------------ | -------------------- | ---------------- |
| `giants`          | 読売ジャイアンツ         | `kashima`            | 鹿島アントラーズ |
| `tigers`          | 阪神タイガース           | `urawa`              | 浦和レッズ       |
| `hawks`           | 福岡ソフトバンクホークス | `kashiwa`            | 柏レイソル       |
| `dragons`         | 中日ドラゴンズ           | `fctokyo`            | FC東京           |

ぜんぶの一覧は `/api/npb/teams` と `/api/jleague/teams` で見られる。

## 8. タイプやリーグで一覧を出す

```html
<poke-card src="/api/pokemon?type=ほのお" title="ほのおタイプ"></poke-card>
<team-card
  src="/api/npb/teams?league=セ・リーグ"
  title="セ・リーグ"
></team-card>
```

ポケモンのタイプ: ほのお・みず・でんき・くさ・こおり・かくとう・どく・
じめん・ひこう・エスパー・むし・いわ・ゴースト・ドラゴン・あく・はがね・
フェアリー・ノーマル。

## 9. 「バックエンドをのぞく」で自分でアドレスを試す

右上の「**バックエンドをのぞく →**」を押すと、別タブでバックエンドの
練習ページが開く。アドレス (例: `/api/pokemon/25`) を書きかえて「送信」を
押すと、バックエンドが返すデータがそのまま見られる。レシピに出てくる
アドレスを ここで先に試してから、自分のページに貼るのもよい。

下の「📡 このページの通信」パネルにも、自分のページが実際に呼んだ
アドレスが出る。クリックすると同じように別タブで開ける。

---

## 10. (発展) 自己紹介や見出しを増やす

```html
<h1>⭐ わたしのページ ⭐</h1>
<p>ここに自己紹介を書こう！</p>
```

`<h1>` の中の文字や絵文字を変えたり、`<p>` をもう1つ増やして自己紹介を
書き足したりしてみよう。ウィジェットの `<poke-card>` や `<team-card>` を
コピーして、もう1つ増やすこともできる (同じタグを2回書いてOK)。

---

## 11. (発展・時間があれば) 自分で fetch を書く

ウィジェットを使わずに、バックエンドから直接データをもらうこともできる:

```html
<p id="my-pokemon">読み込み中…</p>
<script>
  fetch("/api/pokemon/1")
    .then((r) => r.json())
    .then((p) => {
      document.getElementById("my-pokemon").textContent = p.nameJa;
    });
</script>
```

「バックエンドをのぞく」で試したアドレスを `fetch("...")` の中に入れて、
`p.` の後ろを `nameJa` や `sprite` に変えると、いろいろなデータが取り出せる。
